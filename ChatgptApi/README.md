<h4 align="center">后端用的为java后端开源系统</h4>

## 后端介绍

小程序需配合此套java程序，下面将详细介绍此套系统的搭建过程谢谢！

## 后端所需环境

* java8 - jdk1.8
* mysql5.7
* redis
* maven

## 搭建教学

* 首先后端源码下载下来，用idea源码编辑器打开,只需要修改配置文件中的数据库信息（ruoyi-admin/src/main/resources/application-druid.yml）,如果redis设置了密码需要一同改掉
* 然后直接启动项目即可
